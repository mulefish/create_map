## create_map

For a  geojson based game we need  a map. Finding an google map is difficult. This will will be to add layers to geojson.

## running

Currently there is *zero* backend. It will use localstorage in the browser for persistance. So you will need to supply a webserver. I use: 

python3 -m http.server 3030
